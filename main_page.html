This is Mario's Bioinformatic Practical Project Main Page. Use the menu in the top to acces to the different sites. Have fun!

The present project consists in the develop of bioinformatical tools, using python programming languaje, MySQL tables, html and css languages and CGI scripts to query specific information about a microarray dataset of an experiment in C.elegans in which the samples were exposed to cadmium low concentrations during 4 and 24h. After that, the expression profile of its whole genome was analised, as said, using a microarray.

Clic in the 'Database' button in the menu to acces to the query page, where you can query the most relevant information about the experiment.

Details about how to do that are also there!
